================================================================================
                         TROUBLESHOOTING
================================================================================

PROBLEM: "node is not recognized"
SOLUTION: Node.js is not installed. Install it from nodejs.org

PROBLEM: "Cannot find module"
SOLUTION:
   1. Open command prompt in backend folder
   2. Run: npm install
   3. Wait for it to finish
   4. Try starting backend again

PROBLEM: "Port 3001 already in use"
SOLUTION:
   1. Open Task Manager (Ctrl+Shift+Esc)
   2. Find all "Node.js" processes
   3. Right-click each â†’ End Task
   4. Try starting again

PROBLEM: "Port 5001 already in use"
SOLUTION: Same as above - kill all node processes first

PROBLEM: Backend window closes immediately
SOLUTION:
   1. Check if simpleServer.js exists in backend folder
   2. If not, use demoServer.js instead:
      node demoServer.js

PROBLEM: Still getting errors
SOLUTION:
   1. Make sure you're in the correct folder (check the path)
   2. Try running: npm install in both ai-engine and backend folders
   3. Restart your computer and try again
