‚ö†Ô∏è TROUBLESHOOTING:
================================================================================

Problem: "Backend keeps going offline"
Solution:
- Make sure you keep the Command Prompt windows open
- Don't close them until you're done using the chatbot
- Run KEEP-RUNNING.bat for automatic restart

Problem: "Password not saving in Chrome"
Solution:
- Make sure you click "Login" button (not just press Enter)
- Chrome will prompt "Save password?" - click "Save"
- Next time, Chrome will auto-fill your credentials

Problem: "AI not responding"
Solution:
- Check that BOTH services show green status
- Refresh the browser page
- Try toggling to Direct Chat mode (üí¨ icon)

Problem: "Port already in use"
Solution:
- Open Task Manager (Ctrl+Shift+Esc)
- Find "Node.js" processes
- End all Node.js tasks
- Restart the services
